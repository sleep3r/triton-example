## Install:

```bash
make install
```

## Start Services:

```bash
make server-start
```

## API:

Start the FastAPI service:

```bash
PORT=1303 make fastapi
```

Start the FastAPI service locally:

```bash
PORT=1303 make fastapi-local
```
